# JAVA 

Приложение на Java с тремя алгоритмическими задачами.

---

## 1. Быстрая сортировка
- Анимация на Canvas (столбцы).
- Подсветка: pivot – красный, swap – зелёный, готово – золотой.
- Кнопки: **Перемешать**, **Сортировать**.

## 2. WorkerThread
- Поток с `BlockingQueue`. Задачи выполняются последовательно.
- В простое поток **не жрёт CPU** (блокируется).
- Индикатор (работает / спит), лог-консоль, кнопки добавления задач.

## 3. Алгоритм Хаффмана
- Сжатие текста: частота → дерево → бинарные коды.
- Вывод: таблица частот, сжатая строка, коэффициент сжатия.
- Кнопки: **Сжать**, **Декодировать**.

---

## Технологии
- Java, Android SDK (API 24+)
- Material Design, ViewBinding
- Canvas, RecyclerView, BlockingQueue
